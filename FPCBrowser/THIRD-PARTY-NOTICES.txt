## External runtime requirement

`FPCWebBrowser` requires Microsoft Edge WebView2 Runtime on the target machine. The Runtime can be the system Evergreen Runtime or a Fixed Version Runtime selected through `BrowserExecutableFolder`.

The WebView2 Runtime itself is not included in this directory. Redistribution of an Evergreen installer/bootstrapper or a Fixed Version Runtime is outside this package and is governed by the applicable Microsoft Edge WebView2 Runtime terms and notices.

Microsoft distribution documentation: https://learn.microsoft.com/en-us/microsoft-edge/webview2/concepts/distribution

Microsoft WebView2 download page: https://developer.microsoft.com/microsoft-edge/webview2
